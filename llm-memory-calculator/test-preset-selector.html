PresetSelector 优化功能测试

任务 19 实现的功能：

  • 模型搜索功能 - 支持按名称、描述和标签搜索
  • 类别筛选器 - 支持按模型类别过滤
  • 热门模型切换开关 - 可以只显示热门模型
  • 参数范围筛选滑块 - 按参数数量筛选
  • 排序功能 - 按名称、参数数量、类别排序
  • 搜索防抖 - 提高搜索性能
  • DeepSeek模型支持 - 包含特殊标签和推荐用例
  • 响应式设计 - 适配移动设备

测试说明：

1. 在搜索框中输入 "DeepSeek" 或 "推理" 测试搜索功能

2. 点击不同的类别标签测试筛选功能

3. 切换"只显示热门模型"开关

4. 拖动参数范围滑块测试筛选

5. 点击排序按钮测试排序功能

6. 查看DeepSeek模型的特殊标签和推荐用例
